2. Running Code Examples¶

Throughout this books we provide frequent code snippets with the objective of making direct links between mathematical concepts and practical demonstrations. While we endeavour to provide clear and efficient implementations, the code is not intended to be highly optimised nor does it include all of the subtleties required for production ready software.

2.1. Running the Code Live¶

Clicking the rocket logo on the top right of a given page

will provide a drop-down menu with options to launch the page as a Jupyter notebook on Binder or Google Colab. A third option labelled “Live Code” will use TheBe with a Binder backend to enable the code cells to be executed directly within the page.

2.2. Interactive Notebooks¶

Some pages provide the option to launch an “interactive version” of the notebook on Binder. Here “interactive” refers to the use of interactive ipywidgets. These alternative notebooks can be launched by clicking on the Binder badge on the given page.

2.3. Running the Code Locally¶

Alternatively, all of the relevant pages can be downloaded as Jupyter notebooks using the download button on the top right.
